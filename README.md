# Contabilidad Gerencial - Tutor de examen

Herramienta HTML interactiva para estudiar Contabilidad Gerencial.

## Contenido inicial

- Unidades tematicas UT1, UT2/3, UT3 y UT4.
- Lectura obligatoria de Amaro Yardin integrada en UT2/3.
- Practica guiada paso a paso.
- Banco de preguntas multiple opcion.
- Simulacro de examen.
- Progreso guardado localmente en el navegador con `localStorage`.

## Uso

Abrir `index.html` en el navegador o publicar el repositorio con GitHub Pages.
